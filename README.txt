How to install:
1) Open cmd in this folder.

2) npm start.
3) App is running. You can start querying using a tool like Postman (https://www.getpostman.com/).

Queries for example:
GET http://localhost:3000/campaigns?user_id=2786
GET http://localhost:3000/campaigns?user_id=1337

How to test:
1) Open cmd in this folder.
2) npm test.
